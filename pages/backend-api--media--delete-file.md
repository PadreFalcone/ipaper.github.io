## Description

Deletes a given directory.

## Parameters

| Name   | Value
|--------|-------------------------------
| fileID | 	The ID of the file to delete

## Sample Response

```xml
<?xml version="1.0" encoding="utf-8"?>
<result>
  <code>
    <![CDATA[ SUCCESS ]]>
  </code>
  <message></message>
</result>
```

## Return Codes

* SUCCESS
* ERROR
* LOGIN_SECURITY_VIOLATION
